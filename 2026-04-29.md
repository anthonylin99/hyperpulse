# 2026-04-29 HyperPulse Handoff

## Summary
This pass cleaned up the Markets asset detail drawer so the chart is the main surface again. The focus was reducing noisy support/resistance UI, hiding stale levels from the default view, and making the Market Radar cadence clearer for users.

## Shipped Changes
- Market Radar now explicitly says it refreshes every 2 minutes and shows the latest update time.
- Price chart header was simplified by removing the `Green Support`, `Red Resistance`, `Anti-Repaint`, and `Live Fallback` pills.
- Price chart height was increased inside the Markets asset detail view so candles are easier to read.
- Support/resistance sections were renamed to `Support` and `Resistance`.
- Expired and broken levels are filtered out of the default chart/cards.
- Level cards now show distance from current price and touch count instead of internal status labels.
- Trade plan copy now uses `Confirmation level` instead of `Trigger`.
- Mid-range/no-setup states now show `No trade yet`, with targets and invalidation deferred until confirmation.

## Notes For Next Engineer
- Market Radar V1 still refreshes every 2 minutes and can change on each refresh. We clarified this in the UI but did not add hysteresis yet.
- If users still find radar too jumpy, add a small stability buffer: only flip displayed leaders after two consecutive matching refreshes or a clear performance edge.
- Support/resistance remains closed-candle based with DB levels preferred when available and calculated fallback otherwise. The UI intentionally avoids exposing implementation words like `fallback`, `stored observations`, or `expired`.
